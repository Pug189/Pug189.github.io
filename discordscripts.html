Menacing Typing
By Pug#8082
- - - - -

Menacing Typing is a script for Discord.
It notifies you when someone is typing so you can start typing back before they finish their message, or say things like "Type faster"

All you need to do is run the script and go to the Friends screen or a DM so that the sidebar is visible.

If you want it to work elsewhere, run the script on the web version of Discord and keep that one on Friends.
You can then use the app like you usually would, but you'll still get typing notifications.
Copy everything from the text box, go to Discord, CTRL+SHIFT+i, Console, slap everything in there, and then Enter. Then it works until Discord restarts.
